博客测试题

1.ls的用法

ls 查看当前目录下的文件(不包括隐藏文件)

ls -a 查看当前目录下文件(包括以.开头文件例如.bashrc)

ls -l 查看当前目录下文件详细信息
2.mv的用法

mv为move缩写 用于重命名和移动文件

#将1.txt重命名为2.txt

$ mv 1.txt 2.txt

#将1.txt移动至xxx目录

$ mv 1.txt ~/xxx
3.cat的用法

#查看该文件内容

$ cat 文件路径
4.touch的用法

#创建文件

$ touch [dir]

5.explainshell使用

进入explainshell.com 输入想要查询的用法 按下enter即可
